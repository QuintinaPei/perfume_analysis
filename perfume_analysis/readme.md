# 香水成分与受欢迎程度的相关性分析

## 项目概述
本项目基于2024年十大香水的数据挖掘和文本分析，研究香水成分与受欢迎程度之间的相关性。通过消费者评论的情感分析、主题建模和机器学习模型预测，探索哪些香水成分组合更受消费者喜爱。

## 功能模块

### 1. 数据预处理
- 从Word文档中提取香水评论
- 合并多个Excel文件中的数据
- 文本清洗：删除数字、动词、人称代词、副词
- 文本预处理：小写转换、URL移除、非英文字符移除、停用词移除、词形还原

### 2. 情感分析
- 使用VADER情感分析器进行情感分类
- 将评论分为正面、负面、中性三类
- 生成情感分布统计和可视化

### 3. 主题建模
- 使用LDA（隐狄利克雷分布）进行主题建模
- 分析消费者评论中的主要话题
- 计算主题一致性分数

### 4. 香水成分分析
- 分析最受欢迎的单个香水成分
- 分析最受欢迎的香水成分组合
- 分析最不受欢迎的单个香水成分和组合
- 基于情感的词云可视化

### 5. 机器学习模型
- LSTM模型：用于情感态度预测
- SVM模型：支持向量机分类
- 决策树模型：分类预测
- 模型评估：准确率、混淆矩阵、ROC曲线、AUC值
